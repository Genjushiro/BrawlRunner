Brawl Runner

Application disponible sur Windows et Windows Phone

    • Découvrez Brawl Runner!

    Le runner beat-them-all Retro!

    • John le karatéka doit rentrer chez lui dans une ville dangereuse! Aidez le dans sa quete pleine de dangers!

    • Évitez les punks et battez vous contre des hordes de punks en évitant les obstacles pour aller le plus loin possible et peut-etre vous inscrire dans le palmares des meilleurs joueurs de Brawl Runner!

    • Créé par David Sargento, Yvane Jalet et Johnny Yoeurng, Brawl Runner est une application mélangeant un runner classique avec un design retro inspiré par les jeux d'arcade! Brawl Runner mélange les mécaniques d'un runner avec celles d'un beat-them-all classique, battez vous comme un vrai guerrier de la nuit!
